About HealthRisk AI

This web app helps users understand their health risks based on personal data like age, blood pressure, cholesterol, and lifestyle habits.

It uses a machine learning model built with PyTorch, served via Flask, and designed with Bootstrap. The model evaluates inputs and returns a risk level with optional recommendations.

Created with ❤️ by a young dev passionate about machine learning and health tech. Check out the source code on GitHub.
